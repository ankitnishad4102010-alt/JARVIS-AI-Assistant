# 🤖 J.A.R.V.I.S 18.0 - Elite Stark Edition
**Developed by: Ankit Nishad**

J.A.R.V.I.S (Just A Rather Very Intelligent System) is a high-end personal AI assistant built using Python. It features a futuristic Stark Industries-inspired UI with 6 rotating Arc Reactors and is powered by Google's latest Gemini 2.5 Flash AI.

---

## ✨ Key Features

### 🧠 Intelligent Brain
- **Powered by Gemini 2.5 Flash:** Context-aware conversations in smart Hinglish.
- **Neural Voice Engine:** Uses `Edge-TTS` for ultra-clear, human-like Indian accent (Hinglish).

### 🖥️ Futuristic G.U.I (Arc Reactor Mainframe)
- **6 Dynamic Rotating Reactors:** Real-time animated UI representing different system vitals.
- **System Vitals Dashboard:** Live monitoring of CPU Load, RAM Usage, Disk Space (128GB SSD focus), and Battery status.

### 🦾 System Command & Control
- **App Launcher:** Open any app like WhatsApp, Chrome, VS Code, or Calculator via voice.
- **Network Control:** Toggle WiFi ON/OFF (Admin mode) and check Network status.
- **Hardware Integration:** Control Bluetooth settings and system volume.
- **Quick Actions:** Instant System Shutdown, Google Search, and News updates.

### 🕵️ Hacker's Toolkit Integration
- **Nmap Integration:** Capable of network reconnaissance and port scanning.
- **Metasploit Ready:** Designed to handle remote sessions and payload management.

---

## 🛠️ Tech Stack
- **Language:** Python 3.12+
- **UI Framework:** Tkinter (Custom Canvas Animation)
- **AI API:** Google Generative AI (Gemini)
- **Speech:** SpeechRecognition (Google API) & Edge-TTS
- **Libraries:** `psutil`, `pyautogui`, `requests`, `pygame`, `pywhatkit`
